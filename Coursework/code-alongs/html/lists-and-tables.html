How To Make Changes To A Repo With A Pull Request

  1. Use git checkout -b "branchname" to create a branch to make your changes on.
  2. Make your changes to the code and follow the steps for committing them.
    • ⚠️ Make sure to give your commits sensible messages!
  3. Use git push to push the changes to your remote branch.
  4. Go to the repo on Github and make a pull request by clicking the Compare and Pull Request button. Add your title and body.
    • If you have the Github command line tool, you can also use the following command:
      gh pr create --title "prtitle" --body "prbody"
    • ⚠️ Make your PR titles short but descriptive, treat them like commit messages. You can add additional details about the work you've done in the PR body.
  5. Submit your pull request and await for the repo owner to review and/or merge the changes.
